Testing in IE

If you're not on a Windows machine, testing in IE is going to be at least a little inconvenient. You'll need to set up some virtualization software: VMWare, Parallels, and VirtualBox are all good options.

Next, head to modern.IE and download all the virtual machines you think you'll need. It's pretty safe to start with IE 9, 10, & 11. Depending on the project, you may need to get IE 8 as well.

In addition to virtual machines, Microsoft offers a remote testing service that's a bit less involved than installing virtual machines. As of this writing, the remote service only offers IE 11 Technical Preview. Hopefully they'll offer additional IE versions through this service in the near future.

Other Browsers

Safari and Opera are much less used than Firefox & Chrome, but deserve attention nonetheless (particularly Safari). If you're not on a Mac, I'm not sure how you can test in Safari.
